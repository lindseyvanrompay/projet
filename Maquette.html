Fruits and fries est un site web avec une base de données uniquement dédié à l'alimentation de ses utilisateurs.Sur ce site, c'est votre santé qui est la priorité.Pour cela, différentes fonctionnalités existent :

Pour les utilisateurs :

  • La saisie de repas effectués
  • La création de menus et recettes
  • Le suivi de votre alimentation
  • L'abonnement à un ou plusieurs diététicien(s)
  • L'accès à des conseils alimentaires vous étant destinés fournis par un diététicien

Pour les diététiciens :

  • L'accès des statistiques sur l'alimentation d'un groupe d'utilisateurs, vous permettant de rester informer sur l'alimentation de ce groupe
  • La saisie de conseils alimentaires pour guider les utilisateurs vers une meilleure alimentation
